# SeqSieve Roadmap

SeqSieve focuses on exact deduplication with transparent provenance.

## Near-Term

- More parser diagnostics for edge-case FASTQ records.
- Larger browser stress-test fixtures.
- Optional export presets for common workflows.
- Improved keyboard shortcuts for power users.
- More explicit memory usage guidance for large files.

## Scientific Workflow Ideas

- Presets for phage protein database cleanup.
- Presets for HMM input preparation.
- BLAST hit cleanup workflow notes.
- Amplicon dereplication caution templates.
- More IUPAC alphabet validation tests.

## Not Planned as Core Behavior

- Similarity clustering by percent identity or coverage.
- Server-side uploads.
- Tracking or analytics.
- Hidden network calls.

For approximate clustering, use tools such as CD-HIT, MMseqs2, VSEARCH, or usearch-like workflows.
